Etapa 2

Node e NPM

O que é o Node
Node.js é um runtime de JavaScript, criado pelo desenvolvedor Ryan Dahl em 2009. o Node.js, foi
desenvolvido em cima do motor JavaScript V8 - engine criada pelo Google e utilizada nos navegadores
chrome.

O que é NPM
O NPM (Node Package Manager), é o gerenciador de pacotes padrão do node. Ele é utilizado como gerenciamento
de pacotes, fluxo de trabalho em linguagens de programação e ferramenta para construção de front-ends em 
aplicativos e websites.
