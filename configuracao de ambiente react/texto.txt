O que é o Visual Studio Code?
O Visual Studio Code é um editor de código-fonte desenvolvido pela Microsoft para Windows, Linux e
macOS. Ele inclui suporte para controle de versionamento, realce de sintaxe, snippets e refatoração
de código.
